# simple_chat
##### How to run it:
1. Run server ```python server.py```
2. Open chat in browser: http://127.0.0.1:5000/
3. Check number of messages http://127.0.0.1:5000/messages/count

##### How to test it:
```python -m unittest test_app.py```
